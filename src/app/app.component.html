¿Te gustaría ser el personaje principal de un videojuego? ¿Ser el mejor detective de todos los tiempos? ¡¡Con Crime of Murders, el nuevo videojuego web RPG, puedes serlo!!

Crime of Murders es una RPG on-line que adoptará la forma de un escape room. Ustedes jugadores podrán sumergirse en una experiencia donde deberán resolver el enigma para poder escapar.
El objetivo será brindaros una experiencia interactiva envolvente que combine elementos de narrativa, resolución de acertijos y toma de decisiones, todo ello dentro de la página web accesible para diferentes dispositivos.
